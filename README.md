## Overview

This repository is a centralized collection of course materials gathered from personal notes, instructor-provided resources, and contributions from fellow students. It is maintained as part of my role as a **Peer Advisor** at Özyeğin University, with the goal of helping students navigate their coursework more effectively.

The repository is a living archive — materials are added on a rolling basis as they become available, and coverage will expand to include the full OzU Computer Science and AI & Data Engineering core curriculum over time.

---

## What's Inside

Each course folder may contain any combination of the following, depending on what has been collected:

| Type | Description |
|---|---|
| 📖 Lecture Notes & Slides | Instructor-provided or student-compiled notes |
| 📝 Past Exams & Quizzes | Previous semesters' assessments for practice |
| 📂 Homework & Solutions | Problem sets and worked solutions |
| 🤝 Peer Resources | Materials shared by fellow students |
